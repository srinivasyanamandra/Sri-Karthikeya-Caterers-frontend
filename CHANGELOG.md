# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Srikarthikeya Caterers website
- Home page with hero section and service previews
- Dedicated pages for Services, Menus, Gallery, Reviews, About, and Contact
- Entry animation with sacred text
- Responsive design for all devices
- Contact form with validation
- Review filtering system
- Menu filtering by cuisine type
- Vegan menu option
- Design token system with CSS variables
- SEO optimization with meta tags and structured data
- Accessibility features (WCAG 2.1 AA)
- Environment variables configuration
- Code quality tools (ESLint, Prettier, EditorConfig)
- Comprehensive documentation
- Security policy
- Contributing guidelines

### Changed
- Updated all images to show only vegetarian food
- Standardized color scheme with design tokens
- Improved mobile responsiveness

### Security
- Implemented form validation
- Added environment variables for sensitive data
- XSS protection via React
