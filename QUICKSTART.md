# 🚀 Quick Start Guide - Srikarthikeya Caterers Website

## ✅ Installation Complete!

Your React.js website mockup is ready. Follow these simple steps to run it:

## 📦 Step 1: Navigate to Project
```bash
cd srikarthikeya-caterers
```

## 🎯 Step 2: Start Development Server
```bash
npm start
```

The website will automatically open in your browser at `http://localhost:3000`

## 🔧 Customization Tips

### Change Images
In `src/App.js`, find image URLs and replace with your own:
```javascript
image: 'YOUR_IMAGE_URL_HERE'
```

### Update Content
Edit arrays in components:
- `services` array for services
- `menus` array for menu items
- `testimonials` array for reviews
- `team` array for team members

### Change Colors
In `src/App.css`, update CSS variables:
```css
:root {
  --saffron: #E79A2E;
  --deep-green: #1B5E20;
}
```
